# Authometry Homebrew tap

Install the [Authometry](https://github.com/jiayangc1/authometry) configuration CLI on macOS or Linux:

```bash
brew install jiayangc1/tap/authometry
```

Then verify the installation:

```bash
authometry --version
authometry --help
```

Upgrade to the latest release with:

```bash
brew update
brew upgrade authometry
```
